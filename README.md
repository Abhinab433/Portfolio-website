# Abhinab Shrestha — Portfolio

Personal portfolio of **Abhinab Shrestha**, Penetration Tester &amp; Cyber Security Researcher
based in Lalitpur, Nepal.

Built with HTML, Bootstrap 5, AOS, and a custom dark "red-team" theme.

## Sections
- **Home** — intro, typed roles, live terminal card
- **Expertise** — web, mobile, API pentesting & bug bounty
- **Arsenal** — tools, languages, and security frameworks
- **About** — experience, education & community involvement
- **Research** — focus areas and project links
- **Contact** — message form (FormSubmit) and social links

## Run locally
```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
